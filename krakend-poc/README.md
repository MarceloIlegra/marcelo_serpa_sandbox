# Kraken POC


### Book API

Prepare api
```
    cd books-api && npm init && cd ..     
```

Starting book api
```
    node books-api/index.js 
```

### KrakenD

Download KrakenD and extract files inside gateway directory.

Starting KrakenD
```
    ./gateway/usr/bin/krakend run -c gateway/etc/krakend/krakend.json
```
